# Gitワークフロー規約

## ブランチ命名

| 種別 | 形式 | 例 |
| ---- | ---- | -- |
| 新機能 | `feature/xxx` | `feature/capture-loop` |
| バグ修正 | `fix/xxx` | `fix/diff-detection` |

## 実装完了時のPR説明生成

ブランチ上での実装が完了したら、`tmp/pr-description.md` を生成する。
このファイルはgitignoreされており、ユーザーが手動でコピペしてPRに貼り付ける。

### 生成内容のフォーマット

```markdown
## 概要

<!-- 何をしたか1〜2行で -->

## 変更内容

- 変更点1
- 変更点2

## 動作確認

- [ ] 確認項目1
- [ ] 確認項目2

## 備考

<!-- 補足事項があれば -->
```
